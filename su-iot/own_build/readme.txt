In order to run Fritzing container:

1. Make sure your system has X-Server (X11/Xorg) application installed. If not, install it.
    a. for MacOS install [XQuartz]: https://www.xquartz.org/ OR brew install --cask xquartz
    b. for Windows [Xming]: http://www.straightrunning.com/XmingNotes/
    c. for Linux [Xorg]: sudo apt-get install xorg

2. Restart your PC after installing X11 application.
